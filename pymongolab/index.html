PyMongoLab¶

PyMongo-flavored package for accessing to MongoLab databases via MongoLabClient.

Ascending sort order.

pymongolab.DESCENDING = -1¶

  • connection – Connecting to MongoLab
  • database – Database level operations
  • collection – Collection level utilities for MongoLab
  • cursor – Cursor class to iterate over MongoLab query results
