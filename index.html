About

PyMongoLab is a Python_ client library that contains tools for accessing to MongoLab databases via MongoLab REST API_ using a similar sintax to PyMongo_.

from pymongolab import Connection con = Connection("MongoLabAPIKey") db = con.database col = db.collection.find() list(col) [{u'_id': ObjectId('50243d38e4b00c3b3e75fc94'), u'foo': u'bar', u'tld': u'com'}, {u'_id': ObjectId('50004d646cf431171ed53846'), u'foo': u'bar', u'tld': u'org'}]
